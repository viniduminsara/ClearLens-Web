# 🕶️ ClearLens - Frontend

ClearLens is a modern **spectacles e-commerce platform** built with **React**, **Tailwind CSS**, and **Axios** for API handling. It features **JWT authentication** for secure user access and integrates the **PayHere payment gateway** for seamless online transactions.

## Features

**Modern UI** - Styled with Tailwind CSS  
**API Handling** - Axios for efficient API requests  
**JWT Authentication** - Secure login & session management  
**Shopping Cart** - Add, remove, and update items  
**PayHere Payment Integration** - Secure online payments  
**Fully Responsive** - Works on all devices

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **API Handling:** Axios
- **Authentication:** JWT
- **Payment Gateway:** PayHere
